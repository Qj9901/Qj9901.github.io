GitHub Pages 被中间人攻击
发表于 2020-03-26| 更新于 2020-12-23|闲谈
字数总计:272|阅读时长: 1 分钟
| 阅读量:

随便唠唠

听闻国内有人发动中间人攻击，其中 GitHub Pages 和京东最受影响。

京东什么的最近几天没剁手，倒是没什么感觉。GitHub Pages 就不一样了啊，像本博客就是托管在GitHub Pages 上的。

听闻 GitHub Pages 受到了影响，吓的我一激灵，赶紧进去一看，欸，好像没出现什么问题啊。忽然间，灵台似有一道清光席过，恍然大悟。国内托管在 Coding Pages 啊。

于是乎为了验证 “听闻” 是否正确，随便找了个 GitHub Pages 一看，果然啊，上不了了。

如此一来，幸好自己这个小破站是两地托管的。不然。。。哎，不然也没啥啊，反正也没人看不是。

走了走了，就这样吧，随便唠唠。

Ps：刚才传图的时候发现怎么回事传也传不上，突然想起中间人攻击这事。给自己加了个代理马上就传上去了。可怕我刚才还不断登录，可怕可怕，赶紧改密码去了。走了走了~~~
